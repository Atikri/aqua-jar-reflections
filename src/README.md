
# AQUA IN JAR 博客网站

## 如何发布新的博客文章

要发布新的博客文章，请按照以下步骤操作：

1. 打开 GitHub 仓库中的 `src/data/blogPosts.tsx` 文件。

2. 在 `blogPosts` 数组中添加一个新的博客文章对象。确保给新文章一个唯一的 `id`（通常是数组中的最大 id + 1）。

3. 填写以下必要字段：
   - `id`: 唯一标识符（数字）
   - `title`: 文章标题
   - `excerpt`: 文章摘要（显示在博客列表中）
   - `content`: 文章内容（支持 HTML 格式）
   - `date`: 发布日期（格式：YYYY-MM-DD）
   - `author`: 作者名称
   - `tags`: 标签数组
   - `category`: 分类名称

4. 文章内容可以使用 HTML 格式，支持标题、段落、列表、代码块等元素。

5. 示例：
```typescript
{
  id: 7,
  title: "我的新博客文章",
  excerpt: "这是一篇新的博客文章...",
  content: `
    <p>这是我的新博客文章的内容。</p>
    
    <h2>第一部分</h2>
    <p>这是第一部分的内容。</p>
    
    <h2>第二部分</h2>
    <p>这是第二部分的内容。</p>
  `,
  date: "2023-06-01",
  author: "tikri",
  tags: ["新标签", "博客"],
  category: "随笔"
}
```

6. 添加完成后，提交并推送更改到 GitHub 仓库。

7. 网站将自动更新，显示你的新博客文章。

## 注意事项

- 确保每篇文章都有一个唯一的 `id`。
- 日期格式应为 `YYYY-MM-DD`。
- 文章内容支持 HTML 格式，可以包含标题、段落、列表、代码块等元素。
- 图片可以通过 HTML 的 `<img>` 标签引用，但需要确保图片已上传到适当的位置。
